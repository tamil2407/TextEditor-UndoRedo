# Text Editor with Undo/Redo

## Project Description

This project implements a simple console-based text editor using Java. It demonstrates the implementation of Undo and Redo functionality using two stacks. The application follows object-oriented programming principles and serves as a Low-Level Design (LLD) project.

## Features

* Type text
* Undo changes
* Redo changes
* Display current text
* Console-based interface
